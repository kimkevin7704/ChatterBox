To add multiple users:
	1. run program
	2. type user name and click "enter chatroom" button (keep window open to add more users)
	3. chatroom window will open for user entered
	4. go back to the first window - type a new user name and click "enter chatroom"
		this should create a new chatbox for different user

each message submission will update all chatboxes for all users
each message shows who sent the message

this program should support as many users as necessary
